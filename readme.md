# Frontend Revision – Day 1

## Goal
Start learning Frontend Development with strong basics in HTML, CSS and JavaScript.

---

## What is Frontend?
Frontend is the **user interface (UI)** of a website or app – everything that users can see and interact with.

It is mainly built with:
- HTML – Structure
- CSS – Styling
- JavaScript – Functionality

---

## HTML Revision

HTML = HyperText Markup Language

Used to create the structure of a webpage.

### Important Terms:

- **Tag** → `<h1></h1>`
- **Element** → `<p>Hello</p>`
- **Attribute** → `<img src="image.jpg">`

### Most used tags:

- `<html>` – main container
- `<head>` – metadata
- `<body>` – visible content
- `<h1>...<h6>` – headings
- `<p>` – paragraph
- `<a>` – link
- `<img>` – image
- `<div>` – block container
- `<span>` – inline container
- `<ul> <li>` – list
- `<input>` – form field
- `<button>` – button

### Semantic tags (important for SEO):

- `<header>`
- `<footer>`
- `<main>`
- `<section>`
- `<article>`
- `<nav>`

### Types of CSS:

1. Inline CSS

3. External CSS ✅ (Best practice)

### Basic CSS syntax:
